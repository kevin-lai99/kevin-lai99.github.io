Inline VS Inline blocks

With CSS, we can adjust the display properties of most elements on our website. Using Inline and Inline block, we can display elements next to each other within the same line.

Inline

"display:inline" allows us to insert elements in the same line, they do not force the text after them to a new line. Any width and height properties added will have no effect.

Inline-Block

"display:Inline-Block" is similar to "display:Inline" for the most part, however it allows us to alternate the width and height properties of the element.
